# GM - Article

Backend publicador de artículos multi empresa / rubro

## Empezando

### Prerrequisitos

- Java Standard Edition 17
- Gradle 8+

### Instalación

```
$ git clone https://git.../gm-article
$ cd gm-article
$ ./gradlew build
```

## Ejecución de las pruebas

```
$ ./gradlew test
```

## Despliegue

```
$ export SCOPE=local
$ java -jar ./build/libs/application.jar
```
